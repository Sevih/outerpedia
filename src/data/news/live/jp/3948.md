 # 属性：水属性

 # 戦闘タイプ：防御型

 # サブクラス：スイーパー

![](/images/news/live/jp/3948-20a63265.webp)

2) シーズナルスカウトを行った回数に応じて、ゲーム内で以下の表のように報酬が配付されす。

![](/images/news/live/jp/3948-2d055365.webp)

4) スカウト期間終了後に残ったシーズナルスカウトチケットは、次回シーズナルスカウトを行う際に使用できます。 

5) スカウト期間終了後に残ったリミテッド交換Ptは、次回のシーズナルスカウトに繰り越されず、

    【スターピース】に変換されます。(リミテッド交換Pt１につき、スターピース5個)

 6) シーズナルスカウトに登場する新規仲間は、ピースショップ、自己像幻視ダンジョンでピースを獲得することができません。 

7) 別途のスカウト条件がある「デミウルゴス」仲間は、シーズナルスカウトの対象外となります。

8) 【青雲の志 エンバー】は、超越すると専用報酬が獲得できます。

・4★ : プロフィールフレーム ・青雲の志
