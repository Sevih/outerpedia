Hello Masters,

We would like to inform you about several issues that have been identified. Our apologies for the trouble while we work on fixing these issues.”  
  
1\. An issue where Reward Board Points remain in the Hard difficulty rewards of the event dungeon \[Mystic Sage of the Reversed World\]  
→ This issue is currently under investigation.  
  
2\. An issue where the Lucky Charm Shop end date is displayed incorrectly compared to the actual closing date. (Added on 01/02 05:04 AM)  
→ This issue is currently under investigation.

3\. An issue where Stat Modification Locks are not reset when using a Reforge Reset Scroll \[Legendary\]. (Added on 01/02 05:04 AM)  
→ This issue is currently under investigation.  
  
4\. An issue has been identified where the “Draw Omikuji” screen may not be displayed intermittently.  
→ This issue is currently under investigation.  
→ Please note that even if the screen is not displayed, rewards from the “Draw Omikuji” are granted correctly and can be checked in your Inventory.  
  
Once again we apologize for any inconvenience caused.  
We will continue to inform our Masters regarding this issue via this notice.

Thank you.
