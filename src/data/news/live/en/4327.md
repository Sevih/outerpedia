Hello Masters,

We would like to inform you about several issues that have been identified. Our apologies for the trouble while we work on fixing these issues.

![](/images/news/live/en/4327-43ca6af0.webp)

**1\. An issue where the Combine button does not appear in the Special Equipment Enhancement screen when the Mastercraft Enhancement Tool count is 0**

→ This issue is currently under investigation.

**2\. An issue where the Combine button for Skill Manuals and Evolution Stones does not appear under certain conditions**

→ This issue is currently under investigation.

**3\. An issue where the related mission appears to increase by 1 when performing consecutive Reforges**

→ Upon reconnecting, the mission progress is correctly reflected based on the actual number of Reforges performed.

→ This issue is currently under investigation.

**_4\. An issue where Story Normal 3-12 and 4-5 cannot be cleared under certain conditions_**

→ Fixed on 11/18 (Tue) through a patch.

**5\. An issue where certain text in the Core Fusion screen is displayed in Korean**

→ This issue is currently under investigation.

**6\. An issue where the hero’s ★ does not appear and the hero’s level is displayed incorrectly on the Friend Profile, \[Support\], and Arena.** (11/18 10:00 Added)

→ This issue is currently under investigation.

**7\. An issue where, under certain conditions, claiming rewards that include a “Fusion-type Core” triggers an error popup and prevents some rewards from being received.** (11/19 05:30 Added)

→ Fixed on 11/19 (Wed) through a patch.  
– Players affected by this issue will receive the missing rewards.  
– Reward distribution schedule : 11/19(Wed) 12:00 UTC

8\. **An issue where, when the Simplified Ultimate Skill Animation setting is enabled and \[Core Fusion Veronica\] uses her Ultimate Skill, the orinigal Veronica illustration is displayed.** (11/19 06:00 Added)

→ This issue is currently under investigation.

**9\. An issue has been identified where the AP Gain effect intended for Mage-type allies is not displayed or applied when \[Fortuna\] uses her skills.** (11/20 06:15 Added)

→ Fixed on 11/20 (Thu) through a patch.

**10\. An issue where some Guide Quest missions are not being completed** (11/24 08:30 Added)

→ This issue is currently under investigation.

Once again we apologize for any inconvenience caused.

We will continue to inform our Masters regarding this issue via this notice.

Thank you.
